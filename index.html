SgTests

A testing website to show different stylistic choices for the System Green website.

Number 1 has a loading screen, and when all elements of the site are loaded, the screen will rise away.

Number 2 has a loading screen, but when the site is loaded and the screen is gone, there will be keyframe animations on the text and logos.

Number 4 has no loading screen, but has animations on the text and images

Number 3 has no loading screen and no animations on the text or images

Number 5 has no loading screen, but will wait until the website is loaded to play the animations
